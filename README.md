# OneClaw

> 一键安装、无忧享用的个人 AI Agent 平台，专为中国大陆用户设计。

## 特性

- **一键安装** — 全程国内可达，无需 VPN，macOS / Windows / Linux 全平台支持
- **国内模型直连** — 预置百炼、火山方舟、智谱、DeepSeek 等供应商，Coding Plan / API Key / 自定义 endpoint 三种接入方式
- **智能路由** — 模型优先级 + 自动 fallback + API Key 轮换 + 额度追踪
- **成本透明** — 实时用量面板、费用预警、账单导出
- **飞书集成** — Agent 消息通过飞书机器人收发（钉钉、企业微信规划中）
- **内核可替换** — 当前基于 OpenClaw，通过适配层解耦，未来可切换自研内核
